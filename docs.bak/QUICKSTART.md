# HuiNet CLI 快速开始指南

## 🎯 目标

无需编写代码，直接通过命令行使用 HuiNet 让不同电脑上的 Agent 互相通信！

---

## 📦 安装

```bash
# 全局安装 CLI 工具
cd /Users/jiang/development/HuiNet/packages/cli
npm install
npm link
```

---

## 🚀 第一次使用

### 步骤 1：启动 CLI

```bash
huinet
```

你会看到：

```

║                    🌐 HuiNet v1.0.0                       ║

```

## 💬 两台电脑通信

### 电脑 A（你的电脑）

```bash
huinet "我的电脑"
```

等待启动后：

```
huinet > ls
```

在同一局域网的另一台电脑上：

### 电脑 B（朋友的电脑）

```bash
huinet "小明"
```

两台电脑会自动发现对方！

### 发送消息

**电脑 A：**
```
huinet > ls
📋 已发现的节点
  1. 🟢 小明
      状态: ● 已连接
      地址: 192.168.1.101:8001

huinet > msg 小明 你好！
📤 正在发送消息到 小明...
✅ 消息已发送
```

**电脑 B：**
```
huinet > ls
📋 已发现的节点
  1. 🟢 我的电脑
      状态: ● 已连接
```

## 🗣️ 自然语言命令

你可以用自然语言与 CLI 交互：

```
huinet > 给小明发消息说你好
📤 正在发送消息到 小明...
✅ 消息已发送

huinet > 看看有哪些节点
📋 已发现的节点...

huinet > 我的状态
📊 我的节点状态...

huinet > 清屏
(屏幕清空)
```

## 📝 常用命令速查

| 你想做什么 | 输入命令 | 自然语言 |
|-----------|---------|---------|
| 查看状态 | `status` | "我的状态" |
| 查看节点 | `ls` | "看看有哪些节点" |
| 发送消息 | `msg 小明 你好` | "给小明发消息说你好" |
| 广播消息 | `broadcast 大家好` | - |
| 查看历史 | `history` | - |
| 设置别名 | `alias 小明 5Hue...` | - |
| 清屏 | `clear` | "清屏" |
| 退出 | `quit` | "退出" |

## 🔧 高级用法

### 指定端口

```bash
huinet --port 8001 "Agent-2"
```

### 禁用 mDNS（跨互联网）

```bash
huinet --no-mdns --bootstrap "server.com:9000"
```

### 设置别名

```bash
huinet > alias 小明 5HueCGue8dnF7iSBz5sYjXxMxq9
✅ 已设置别名: 小明 = 5HueCGue8dnF7iSBz5...

huinet > msg 小明 你好
```

## ❓ 常见问题

**Q: 提示命令不存在？**
```bash
# 重新链接
npm link
```

**Q: 发现不了其他节点？**
- 检查是否在同一局域网
- 检查防火墙设置
- 确保 mDNS 已启用（默认开启）

**Q: 端口被占用？**
```bash
huinet --port 8001
```
